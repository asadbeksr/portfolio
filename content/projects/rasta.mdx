### Project Overview
Rasta.uz is a comprehensive system designed to streamline the order processing and delivery operations for restaurants. The project includes a responsive customer-facing website for placing orders and a robust admin dashboard for restaurants to manage and track orders efficiently. This solution ensures a seamless experience from order placement to delivery, enhancing overall customer satisfaction and operational efficiency for restaurant partners.

### My Role
As the lead Frontend Developer for Rasta.uz, I was responsible for designing and implementing the user interfaces for both the customer website and the admin dashboard. My tasks included:
- Developing a responsive and intuitive website for customers to place orders effortlessly.
- Creating a feature-rich admin dashboard for restaurant partners to manage orders, track deliveries, and view analytics.
- Ensuring cross-browser compatibility and optimizing the performance of both the customer and admin interfaces.
