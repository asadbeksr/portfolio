### Project Overview
OxBox is a Questionnaire Constructor designed to allow users to create, manage, and analyze questionnaires effortlessly. This tool provides a user-friendly interface for designing custom questionnaires, setting up various question types, and collecting responses. It includes features for managing question logic, analyzing results, and exporting data.

### My Role
As the lead Frontend Developer for OxBox, my responsibilities included:
- Designing and implementing the user interface for creating and managing questionnaires.
- Developing features to support various question types and response collection methods.
- Ensuring the application’s usability and responsiveness across different devices.
- Collaborating with backend developers to integrate frontend components with the system’s core functionalities.

### Skills Enhanced
Working on OxBox allowed me to improve and develop several key skills:
- **UI/UX Design**: Enhanced my ability to create intuitive and user-friendly interfaces for complex forms and data collection tools.
- **Form Management**: Gained experience in developing systems that handle dynamic form creation and response management.
- **Responsive Design**: Strengthened my skills in ensuring that interfaces work seamlessly across a range of devices.
- **Frontend-Backend Integration**: Improved my ability to work with backend systems to deliver a fully integrated solution.
